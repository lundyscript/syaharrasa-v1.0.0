# syaharrasa-v1.0.0
Toko online ini dibuat menggunakan teknologi laravel dan tailwind. Sama seperti ecommerce pada umumnya, toko online ini dapat menambahkan dan menghapus produk dan katagori produk yang ditampilkan pada laman utamanya. Admin juga dapat memberikan diskon pada produk serta produk dapat di atur ketersediaannya. Dalam penggunaannya, sistem dapat membuat invoice secara otomatis agar konsumen dan admin dapat mengetahui rincian pesanannya. Adanya fitur keranjang belanja untuk mempermudah konsumen mengatur produk yang ingin dibeli. Terdapat beberapa status pada pesanan konsumen seperti unpaid, processed, completed, cancelled untuk mempermudah konsumen melacak pesanannya.
